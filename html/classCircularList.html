Detailed Description

template<typename T>
class CircularList< T >

Template class for a circular doubly linked list.

Template Parameters
TType of elements stored in the list

The class implements a circular doubly linked list with support for all basic STL container operations. Each element in the list contains pointers to the previous and next elements, with the last element linked to the first, forming a circular structure.
